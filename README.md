# Import Access logs to Elasticsearch 
---

  - filter access logs
  - import access logs to elasticsearch from file
  - use es.index to create a new index for logs
  - import logs by log time

### Environemnt
 - Elasticsearch Server == 6.3.0 (host by AWS)
 - Python == 3.6
 - elasticsearch == 6.3.0

### Tech

Import logs to Elasticsearch uses a open source modules to work properly:

* [Python Elasticsearch Client](https://elasticsearch-py.readthedocs.io/en/master/) 

### Usage

```sh
$ pip3.6 install -r requirements.txt
$ python3.6 import_access_log.py 
```
